Zasady gry

📋 Jak grać?

1. Zaloguj się swoim nickiem i hasłem (przesłanym przez organizatora).
2. Przejdź do zakładki Mecze & Typowania.
3. Dla każdego meczu wybierz: wygrana drużyny 1, remis lub wygrana drużyny 2.
4. Potwierdź typ przyciskiem Zapisz.
5. Typy można zmieniać do 30 minut przed rozpoczęciem meczu.
6. Po meczu punkty naliczają się automatycznie.

💰 Punktacja

Każdy typ to wirtualny zakład 100 punktów.
Jeśli trafisz wynik → zdobywasz: 100 × kurs punktów.

Przykład: Polska wygrywa @ 2.40 → zdobywasz 240 pkt.
Remis @ 3.10 → zdobywasz 310 pkt.
Błędny typ → 0 punktów.

Kursy są pobierane automatycznie przed meczem.

⏰ Deadline

Ostateczny termin na oddanie/zmianę typu to 30 minut przed godziną kickoffu.
Po tym czasie przycisk zostaje zablokowany.
Nie można obstawiać po starcie meczu!

🔧 Konfiguracja (dla organizatora)

Dane (mecze, kursy, wyniki) są pobierane z football-data.org API i zapisywane w Google Sheets jako baza danych.
Instrukcja konfiguracji Google Apps Script znajduje się w pliku README dołączonym do aplikacji.

Panel admina

⚙️ Konfiguracja Google Sheets API
Wdróż Apps Script jako Web App i wklej URL tutaj
Klucz jest bezpiecznie przekazywany do Apps Script — Apps Script pobiera kursy i wyniki po stronie Google (bez CORS). Zarejestruj się na the-odds-api.com → darmowy plan → wklej klucz.

💡 Tip: Możesz też wpisać klucze na stałe bezpośrednio w kodzie HTML (klucze są już wpisane na stałe w kodzie) — wtedy nie musisz konfigurować po każdym wdrożeniu.
